　第一场雪，在新年的第一天飘落在心里，融化成化不开的忧伤。原来天空一直都是灰色的。 　　那幽幽的蓝，只能在梦里，那暖暖的阳光，只能在梦里，那鸟语花香的春日，也只能是在梦里。 　　轻轻的告诉自己，要坚强。一路走来都是这样一个人，和自己说话，让文字渲泄心底深处的快乐和忧伤。 　　当眼泪不再轻易的流，当心开始寒冷开始麻木，是不是还会有一个温暖的地方让我融化？是不是还要乞求那可怜的一份爱，一份关怀？ 　　零九年的第一天，有一场雪，把我的希望就此埋没。或许，有些东西得到的时候就是失去。 　　有风的日子，让风吹散心中的阴霾吧，再冷的寒冬，我都要勇敢的面对和渡过！允许自己忧伤，允许自己留恋，但是不能允许自己失去骄傲，不能允许自己失去笑容！ 　　来时的路上蝶舞纷飞，破茧成蝶的痛是不是就忘记了呢？痛的时候，躲在自己的文字里，安静而遥远。 　　新年的第一天，心碎成片片雪花飘落在遥远的梦里！无悔的痛着…… 新年第一场雪散文【二】：今冬的第一场雪 　　天深深的隐去秋的颜色，无心的绘出素意的景观。勾列出瘦中有腴的褶皱、沧桑俊美的韵致、层层结迭的雄浑、古朴苍劲混沌的艺术之美！ 　　苍穹阴韵的深沉，雪似乎从时间的隧道脱颖而出，轻盈着无翼之趣！风若远年的触及着红尘轮回之约！是又一春的花开？还是五季的花谢？ 　　飘落之处，灿笑如絮，蒙蒙洞天白一色，飘渺之处逗心魂，雪悄悄拂在脸上，落在掌心，幼嫩的希冀，及至不可辨认的灵性，妩媚流动着安逸之姿，真是“清凉之雪意，淡笑一瞬间”便隐逝了晶莹的羞涩，静静地沁着心事粘连的泪痕，模糊了诱人忘情的猜测！ 　　千朵，万朵，朵朵纯洁无暇之爱，起伏的雪浪旋转着自由的闲暇！看着雪凝绽谢，纷乱如花的梦幻缠绵了潮湿跳跃的音符，渲染了明快寂廖默默执着的心声。在这雪白的世界、惆怅、欢喜，爱与恨都变的轻盈，简单，飘飘羽然，而又渐渐升腾！升腾着寒意丝丝如馨如兰的气息，沉浮在天地间、世俗中、吻痕里悄悄顾盼！ 　　雪是灵魂出窍的禅衣，简洁了世界、寂寞了心声、说轻不轻、说淡不淡的写意，闲散了天地朦胧的唯美！在似与不似之中、在意与无意之间、在飘未飘落、去与未去之间回荡！纯纯的雪意洁白了三岸的风情，三世朦胧、与三心二意的灵性！ 　　山水依晰的脉络，雪冀万物的绽放！虔诚飘陨着如痴如醉的心语，在初雪的大地上无限的延伸，延伸着洁白散漫的深深意境！深深雕琢了不离不弃的融情！ 新年第一场雪散文【三】：2006年的第一场雪 　　今年元旦刚过，一场迎春的小雪及时落下，为人们带来新年的欢乐与祝福。雪下得不疾不徐，悠然自得，先是碎碎的小雪粒，后来雪粒渐渐增大，眼见着雪花开始飘舞起来了。还没等人陶醉畅想片刻，雪花悄然退场，徐徐的雪粒带着些湿气重新撒漫了整个灰色的天空。雪粒落地的声音很小也很轻，有点毛毛雪的意味，走在外面不一会头上、衣襟上满是湿润松散的雪粒，用手拂去很快却又贴身上头，让人颇有些无可奈何。 　　虽然有雪降落，但地温很快将路面上的薄雪融化，树梢上、草坪中、建筑的背阴处还是留下了一层层洁白温润的冬雪，为新年的到来献上一份宝贵的甜美记忆。看不到冰天雪地，望不见玉树琼花，只有这细细碎碎的小雪相伴，一丝惆怅，一点感怀，一寸相思，点点滴滴袭上心头。虽然没有大雪的壮美与浩瀚，但小雪自有其贴心与清凉，再加上空气清新无比，小城也少有喧哗，不需踏雪寻冬自有暗香萦怀，正所谓眼中小雪，心中大雪，洗尽铅华，阅尽冬日雪韵清香。 　　下了一整天的小雪，到夜晚也不见增其厚度，仅仅两三公分而已，漫天的水气笼罩着箭乡大地。待到第二日清晨，地面的水气与高空的寒气相遇，一场朦胧的大雾遮盖住了眼前的一切，能见度不到十米，白天看远处的车灯忽隐忽现，雾中的大树、房屋轮廓也不十分鲜明，不要说雾里看花，看什么都是神龙见首不见尾，天地间呈现出一种近距离的朦胧飘逸美感。 　　持续了一日的大雾天刚过，紧接着连续两日的雾凇气候更增添了箭乡冬日的雪韵寒气。大小树木全部结满了浅浅的雾凇，像施了一层淡妆，远看灰白，近观素白，细看莹白，幕顶垂地，淡雅风情，宁静秀姿，令人心中赞叹不已。寒气尚未逼人，但早有欢乐的少年奔跑于雾凇林间玩耍，用纯洁的眼神捕捉着冬日的寸寸光华美景。等雾凇褪尽，天气并未转暖，依旧灰蒙蒙的，一夜之间第二场小雪仿佛被追赶着再次闯入冬日的宽厚怀抱。脚下的薄雪净白润滑，踩上去咯嘣咯嘣的闷响，听到了冬雪下春天的轻轻呼唤声，陡然间仿佛与春天差点撞个满怀。这个冬日依然不太冷，满怀心事的冬天有些半途而废的羞惭，一些吝啬的小雪是难以满足人们眼中的无限饥渴与期盼的。但走进新年中的人们依然是幸福的，既看到了小小的欢喜，也留下淡淡的遗憾。这遗憾非但没有让人烦恼，反而加浓了幸福中苦尽甘来、醇厚绵长的原生滋味，原来幸福的感受是来自于对美好未来的欢欣期待与自强不息中。 该文章《新年第一场雪散文》来源于出国留学网，
